## Overview

This project covered the basic theory of neural networks. The PINN was constructed using PyTorch. Explanations of the use of PyTorch are minimal but some code blocks are explained throughout the report. The project mostly focussed on the comparison of PINNs to numerical solvers with the (perhaps expected) result that numerical solvers are *quite good*. PINNs are clearly better at certain things, and given more time I'd have liked to have looked using PINNs to identify smoothly varying boundaries (thermal conductivity and such).
